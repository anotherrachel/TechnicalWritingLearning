Get Started

Create a WIzNote account.

  1. Download WizNote from http://www.wiz.cn/download.html.

  2. Run WIzNote and choose a language.
  3. Select the folder where you would like to install WizNote, and click "Next".

  4. Keep selecting "Next" until the finish page appears, and click "Finish" to exit setup.

  5. You can click the button, and sign up for an account.
