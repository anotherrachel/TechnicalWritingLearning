Tags on the Interface

Add tags on the interface.
  1. Right-click "Tags" on the leftmost bar and select "New Tag...".
  2. Input a tag name, for instance, "idea" and click "OK".
  3. Right-click "idea" tag and get its subtag by selecting "New Tag...".
