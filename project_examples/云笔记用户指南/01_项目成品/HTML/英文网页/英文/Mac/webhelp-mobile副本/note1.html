Create a Note Folder

WizNote offers multi-layered folders to help you organize notes.

  1. Right-click "Folders" on the leftmost bar.

  2. Input a name, for instance, "ideas", and click "OK" to get a new folder.

  3. Right-click "ideas" and select "New Folder" to get its subfolder.
